# ArtaPayamak Node.js OTP Example

Node.js example for sending OTP SMS via IPPanel Edge API.

## About

This repository shows a simple example of sending OTP SMS using Axios and IPPanel Edge API.

Website:
https://artapayamak.com

## Requirements

- Node.js 16+
- `axios`

## Setup

Add your API token to `.env`:
```env
IPPANEL_API_TOKEN=your_api_token_here
```
## Install
npm install axios
Example
Main example file:

`send_otp.js`

## Security Note
Do not hardcode your real API token in source code.

Use environment variables instead.
